# eslint-config-poddify

An eslint config for reuse across projects

## Installation

```bash
$ npm i --save-dev @poddify/eslint-config-poddify
```

## Usage

In your `.eslintrc`

```json
{
    "extends": "@poddify/eslint-config-poddify",
    "rules": {
        "your project": "your rules"
    }
}
```
